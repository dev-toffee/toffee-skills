# toffee-skills

Agent skills for [Toffee](https://toffee.at) — AI agent detection for websites.

## Available Skills

### toffee-integration

Guides integration of the Toffee SDK (`@agent-radar/sdk`) into existing or new projects. Covers npm install, script tag, React/Next.js/Vue setup, tracking pixel, element tracking, self-hosted deployment, and common mistakes.

## Install

```bash
npx skills add VishiATChoudhary/toffee-skills@toffee-integration
```

Or install globally:

```bash
npx skills add VishiATChoudhary/toffee-skills@toffee-integration -g -y
```
